## Software Engineer Backend Test 1

## Installation

```bash
$ npm install
```

## Migration

update your mysql config in `env` file

```bash
$ npm run migration:run
```

## Seed
```bash
$ npm run seed
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```


## Swagger URL
http://localhost:3000/api

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
